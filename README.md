# kochava-k4a-mcp-server

Kochava for Advertisers - the Omni-Channel measurement and attribution platform provided by Kochava incorporating last click attribution and incrementality and mmm tools for brands and their agencies. The Kochava Analytics MCP Server facilitates Query mobile attribution data, campaign performance, trackers, segments, and reports. Free App Analytics (FAA) account creation and registration built-in — no Kochava account needed to get started
